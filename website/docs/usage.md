---

After installing the `my-typescript-react-library`, you can start using its components in your React application:

```typescript
import React from 'react';
import { MyComponent } from 'my-typescript-react-library';

function App() {
  return (
    <div>
      <MyComponent />
    </div>
  );
}

export default App;
```
